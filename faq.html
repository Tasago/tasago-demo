Ayuda y preguntas frecuentes

Respuestas claras, sin letra chica.

¿Qué incluye cada plan?

  • Express 24h ($900): 3 comparables + rango de valor. Informe breve (2–3 páginas).
  • Pro PDF (1.5 UF, 3–4 días): 6–10 comparables + ajustes explicados + anexos. 1 revisión incluida.
  • Visita + Informe (desde 2.5 UF, 4–5 días): visita técnica, fotos propias y capítulo de inspección. Tarifa varía por zona/traslado.

¿La zona afecta el precio?

Solo para Visita + Informe (traslado). Express y Pro no cambian por zona.

¿Qué necesito para comenzar?

  • Comuna + dirección
  • Tipo de propiedad, m² aprox., dormitorios/baños
  • Fotos (ideal) y plano (recomendado)

¿Qué es firma simple?

Es una forma de identificación del firmante (nombre/RUT/fecha y evidencia de aceptación). Para este servicio funciona como respaldo práctico del encargo y del informe.

Si necesitas un informe para banco u otro trámite formal, avísanos para ajustar requerimientos.
